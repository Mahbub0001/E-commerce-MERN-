# NovaMart

Full-stack MERN e-commerce app.

Stack:
- React + Vite
- Tailwind CSS
- Framer Motion
- Lucide React
- Node.js + Express.js
- MongoDB + Mongoose
- JWT authentication
- bcryptjs password hashing
- Axios API calls
- Fake checkout and order flow

See [docs/PROJECT_STRUCTURE.md](docs/PROJECT_STRUCTURE.md) for project architecture.
